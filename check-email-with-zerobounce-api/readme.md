# What does it do ? 

The idea of this Custom Coded Action is to user zeroBounce API with HubSpot. 


```
 The output of the Custom Coded Action is : 
┌────────────┬──────────────────────────────┐
│  (index)   │            Values            │
├────────────┼──────────────────────────────┤
│   status   │           'valid'            │

│ free_email │             true             │
│ firstname  │             null             │
│  lastname  │             null             │
│   gender   │             null             │
│  country   │             null             │
│   region   │             null             │

┌────────────┬───────────┐
│  (index)   │  Values   │
```
